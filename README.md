# DDKI

Eine Linux-Desktop-App zum Erfassen, Verwalten und Analysieren von Fahrstrecken – gebaut mit Python und PySide6 (Qt6).

## Features

- Monatsbasierte Fahrtenbuch-Verwaltung
- Automatische Berechnung der Gesamtstrecke (Hin & Zurück)
- Fehleranzeige bei ungültigen Einträgen
- Light- und Dark-Mode
- Automatische Update-Prüfung

## Installation

```bash
git clone https://github.com/DEIN_USER/DDKI.git
cd DDKI
./install.sh
```

## Starten

```bash
./run.sh
```

## Voraussetzungen

- Python 3.11+
- Linux (getestet auf Ubuntu/Debian)
